# Fan Stride Website

Official website for Fan Stride - For the Fan. Fueling the Runner.

## About

Fan Stride brings spectators, athletes, and race directors together through augmented reality. Point your camera at any runner to see their story, support their cause, and cheer them on in real-time.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React Icons

## Deployment

The site is configured to deploy to GitHub Pages. See `DNS-SETUP.md` for domain configuration details.
